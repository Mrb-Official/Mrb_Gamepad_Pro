# MRB Controller Pro

An Android app that turns your Android device into a Bluetooth HID gamepad, tailored for racing games.

## Project Overview

- **Type**: Native Android application
- **Language**: Kotlin
- **Build System**: Gradle (Groovy DSL), Gradle Wrapper 8.11.1
- **Min SDK**: 28 (Android 9.0 Pie)
- **Target SDK**: 34 (Android 14)

## Features

- Bluetooth HID gamepad emulation
- Steering wheel tilt controls (accelerometer/gyroscope)
- Gas/brake pedals, gear shifters, D-pad, face buttons
- 30+ customizable overlay buttons (Premium feature)
- Premium system unlocked via rewarded ads (24-hour access)
- Custom button layout save/load via SharedPreferences

## Dependencies

- `androidx.core:core-ktx:1.12.0`
- `androidx.appcompat:appcompat:1.6.1`
- `com.google.android.material:material:1.11.0`
- `com.google.android.gms:play-services-ads:23.0.0`

## Build Setup

The project requires Android SDK and Java to build. The `build_apk.sh` script handles automatic SDK installation:

1. Downloads Android command-line tools if missing (`$HOME/android-sdk`)
2. Installs `platform-tools`, `platforms;android-34`, `build-tools;34.0.0`
3. Writes `local.properties` with `sdk.dir`
4. Runs `./gradlew assembleDebug`

**Output**: `./app/build/outputs/apk/debug/app-debug.apk`

## Workflow

- **Build APK**: Runs `bash build_apk.sh` — builds the debug APK. First run downloads Android SDK (~5 minutes), subsequent runs are faster.

## Project Structure

```
app/src/main/java/com/mrb/controller/pro/
  MainActivity.kt     - Main UI, sensor input, button handling
  HidService.kt       - Bluetooth HID service
  SplashActivity.kt   - Splash/entry screen
  BtnDef.kt           - Button definitions and data structures
app/src/main/res/
  layout/             - XML layouts
  drawable/           - Button icons and drawables
  values/             - Strings, colors, styles
app/src/main/assets/  - Audio files (boot.mp3, connect.mp3)
```
